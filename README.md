# Rock-Paper-Scissors Game

A simple Rock-Paper-Scissors game built with HTML, CSS, and JavaScript.

## Features
- Play against the computer
- Displays the winner after each round
- Adds cumulative points in conventional Blue and Red color.
- Simple interactive UI with buttons


## How to Run
1. Clone or download the repository.
2. Open `index.html` in your web browser.
3. Click on **Rock**, **Paper**, or **Scissors** to play.
4. The opponent automatically picks one, and the result is displayed immediately.


## Notes
- All game logic is handled in `script.js`.
- This project is meant for learning and practice.
